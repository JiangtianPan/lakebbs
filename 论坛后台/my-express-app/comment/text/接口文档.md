# 后台接口

## 管理员登录
- **通用请求路径前缀：** `http://localhost:8880/api/commerce/v1/`
#### 管理员账号
```json
{
  "usernameOrEmail": "admin",
  "password": "123456"
}
```
## 登录接口
- **请求路径：** /login
- **请求方式：** POST

### 请求参数:
```json
{
  "mobile": "string",
  "password": "string"
}
```
### 返回参数:
```json
{
  "msg": "string",  
  "data": "string"
  "code": 200,
}
```
